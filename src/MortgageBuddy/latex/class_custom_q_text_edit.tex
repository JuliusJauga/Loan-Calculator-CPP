{\em event} & The key event.\\
\hline

This function is called when a key press event occurs. If the Tab key is pressed, it ignores the event and focuses on the next child widget, allowing the user to move between input fields using the Tab key. If any other key is pressed, it calls the base class\textquotesingle{}s key\+Press\+Event function to handle the event.
