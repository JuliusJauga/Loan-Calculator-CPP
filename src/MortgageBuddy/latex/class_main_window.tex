\doxysubsection{Detailed Description}
The \doxylink{class_main_window}{Main\+Window} class represents the main window of the Mortgage Buddy application. 

This class inherits from QMain\+Window and provides functionality for creating and managing the user interface, as well as performing various calculations and operations related to mortgage calculations.
